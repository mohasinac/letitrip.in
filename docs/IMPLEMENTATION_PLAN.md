## Phase 58 ‚Äî Service ‚Üí Hook Migration (Rule 20 Strict Compliance)

**Goal:** Every client component that currently imports a service and passes it inline to `useApiQuery`/`useApiMutation` must be refactored to consume a **named custom hook** instead. After this phase, no client component or page (`"use client"`) may import from `@/services` or `@/features/*/services` ‚Äî only hooks do.

**Pattern being fixed:**

```tsx
// BEFORE (Rule 20 violation ‚Äî service imported in component)
import { productService } from "@/services";
const { data } = useApiQuery({ queryFn: () => productService.list(params) }); // ‚ùå

// AFTER (Rule 20 compliant)
import { useProducts } from "@/features/products";
const { data } = useProducts(params); // ‚úÖ
```
