### Configuration

#### Allowed Areas

- All public pages
- Product listings
- Category pages
- Shop pages
- Legal pages
- FAQ

#### Disallowed Areas

```
/api/*          - Backend APIs
/user/*         - User dashboards
/seller/*       - Seller dashboards
/admin/*        - Admin area
/logout         - Auth pages
/unauthorized   - Error pages
?*utm_*         - UTM parameters
?*ref=*         - Referral parameters
```

#### Crawl Delays

- Googlebot: 0 seconds (no delay)
- Bingbot: 1 second
- Other bots: 1 second

### How to Use Schemas

```tsx
import { generateProductSchema, generateJSONLD } from "@/lib/seo/schema";

export default function ProductPage({ product }) {
  const productSchema = generateProductSchema({
    name: product.name,
    description: product.description,
    // ... other fields
  });

  return (
    <>
      <script
        type="application/ld+json"
        dangerouslySetInnerHTML={generateJSONLD(productSchema)}
      />
      {/* Page content */}
    </>
  );
}
```

## 7. Canonical URLs

### Implementation

Canonical URLs are automatically generated for all pages using the `generateMetadata()` function:

```typescript
alternates: {
  canonical: url,
}
```

### Benefits

- Prevents duplicate content issues
- Consolidates link signals
- Improves crawl efficiency
- Better indexing

### Example

```html
<link rel="canonical" href="https://justforview.in/products/product-name" />
```
